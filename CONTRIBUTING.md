# Contribution Guidelines

## المساهمة في المشروع

شكراً لاهتمامك بمساهمتك في تطبيق شات ليالى زمان!

### كيفية المساهمة:

1. **Fork المشروع**
   - اضغط على Fork في أعلى الصفحة

2. **عمل نسخة محلية**
   ```bash
   git clone https://github.com/[username]/ChatLiyaliZaman.git
   cd ChatLiyaliZaman
   ```

3. **إنشاء فرع جديد**
   ```bash
   git checkout -b feature/اسم-الميزة
   ```

4. **عمل التغييرات**
   - اتبع معايير الكود
   - أضف التعليقات المناسبة

5. **Commit والـ Push**
   ```bash
   git add .
   git commit -m "وصف التغيير"
   git push origin feature/اسم-الميزة
   ```

6. **عمل Pull Request**
   - اشرح التغييرات
   - انتظر المراجعة

### معايير الكود:

- استخدم Kotlin للـ Code الجديد
- اتبع Material Design Guidelines
- أضف التعليقات حيث يكون الكود معقداً
- اختبر قبل الإرسال

### الإبلاغ عن الأخطاء:

1. تحقق أن المشكلة لم تُبلغ مسبقاً
2. اشرح المشكلة بوضوح
3. أرفق صور أو فيديوهات إن أمكن
4. اذكر نسخة Android التي تستخدمها

### الاقتراحات:

- اكتب الاقتراح بوضوح
- اشرح لماذا هذه الميزة مهمة
- قدم أمثلة إن أمكن

### الترخيص:

بالمساهمة، فأنت توافق على أن يكون المشروع تحت ترخيص MIT.

---

شكراً لمساهمتك! 🙏
